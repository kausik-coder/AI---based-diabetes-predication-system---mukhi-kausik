# AI-Based Diabetes Prediction System

## Overview

This project predicts whether a person is diabetic based on health indicators using a Random Forest Machine Learning model. The application is built with Python and Streamlit.

## Features

* Diabetes prediction using Machine Learning
* Random Forest Classifier
* Interactive Streamlit web interface
* Diabetes risk percentage
* Real-time prediction

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Streamlit
* Joblib
* Git & GitHub

## Dataset

Diabetes Health Indicators Dataset (BRFSS 2015)

## Model Performance

* Accuracy: 73.71%
* Precision: 71.72%
* Recall: 78.04%
* F1 Score: 74.75%

## Installation

```bash
pip install -r requirements.txt
python -m streamlit run app.py
```
